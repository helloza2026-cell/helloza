# Helloza Bpo Solutions

Startup-focused BPO services website built with Vite, React, TypeScript, and Tailwind CSS.
Live domain: `https://helloza.co.in`

## SEO and Deploy

- Netlify-ready with `netlify.toml`
- Sitemap: `/sitemap.xml`
- Robots: `/robots.txt`
- Brand metadata set in `index.html`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run test`
